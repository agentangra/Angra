Angra Etsy Draft Tools

Angra Etsy Draft Tools is a private automation utility used by the shop owner to prepare Etsy draft listings from approved digital artwork files. Listings are created as drafts only; the shop owner reviews every draft and publishes manually inside Etsy.

Draft-only The tool prepares draft listings and does not automatically publish or activate listings.
Owner-approved Etsy access is granted only through the shop owner’s OAuth authorization.
Limited purpose The tool uses Etsy data only to create and manage listing drafts requested by the shop owner.
